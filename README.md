# 个人博客系统

这是一个基于 Next.js 和 TypeScript 开发的个人博客系统。

## 技术栈

- Next.js 14.2.5
- TypeScript
- Ant Design 5.19.3
- Bytemd 1.21.0

## 主要功能

### 1. 主题切换

- 支持多主题色切换
- 使用 CSS 变量实现动态主题
- 平滑的过渡动画效果

### 2. 文章管理

- Markdown 编辑器支持
- 文章分类管理
- 文章列表展示
- 文章详情阅读

### 3. 响应式设计

- 适配移动端和桌面端
- 流畅的动画过渡效果

### 4. 文件系统

- 支持文件夹管理
- 支持文件上传
- 支持文件预览
